# API Stores
### Repositorio de la API del proyecto stores

Para comenzar a trabajar en él:

```
cd your-directory-to-work
git clone https://github.com/edwingarces/store_api.git
cd store_api
code ./
```

Para instalar las dependencias:
```
npm i
```
O
```
yarn
```
